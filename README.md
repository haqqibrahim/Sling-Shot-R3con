## Getting Started

### Prerequisites

Make sure to install the required dependencies by running the following script:

```bash
chmod +x ./requirements.sh
./requirements.sh
```

### Usage

Run Sling Shot R3con by providing the target domain as follows:

```bash
./shoot.sh domain.com
```

### Example:

```bash
./shoot.sh google.com
```
